## Requirements
- Ruby 3.x, Rails 7.x
- Node.js 18+
- npm or yarn

## Backend Setup (Rails)
bundle install
rails db:create
rails db:migrate
bundle exec rails server

API runs at: http://localhost:3000

## Frontend Setup (Svelte)
cd frontend
npm install
npm run dev

Frontend runs at: http://localhost:5173


## Running the App
- Start Rails backend (`rails s`)
- Start Svelte frontend (`npm run dev`)
